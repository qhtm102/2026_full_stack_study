HTML Form 실습 - 게시판 글쓰기 화면

[ 목표 ]

HTML Form 태그를 활용하여 게시판 글쓰기 화면을 작성한다.

[ 요구사항 ]

1. 기본 구조
   <form> 태그를 사용한다.
   화면 상단에 "게시판 글쓰기" 제목을 표시한다.
   form의 method는 post로 설정한다.
   form의 action은 사용하지 않는다.

2. 입력 항목

   항목	    입력 방식	    조건
   작성자	text		필수 입력
   비밀번호	password	필수 입력
   제목	    text		필수 입력
   내용		textarea	5줄 이상 입력 가능
   카테고리	select		공지사항, 질문, 자유게시판 중 선택
   공개 여부	radio		공개 / 비공개 중 선택
   첨부파일	file		파일 선택 가능
   작성 동의	checkbox	게시글 등록 동의

3. 버튼

   등록 버튼 (입력한 데이터를 서버로 전송한다.)
   취소 버튼 (입력한 내용을 초기화한다.)

4. 화면 구성 조건

   각 입력 항목에 <label> 태그를 사용한다.
   필수 입력 항목은 * 표시를 한다.
   placeholder 속성을 적절히 활용한다.
   표(<table>)를 이용하여 정렬한다.
   사용자가 보기 쉽도록 항목을 배치한다.

5. 예시 화면

게시판 글쓰기

작성자 * [____________________]

비밀번호 * [____________________]

제목 * [____________________]

카테고리 [자유게시판 ▼]

공개 여부 (●) 공개 ( ) 비공개

내용 *
┌─────────────────────┐
│                     │
│                     │
│                     │
│                     │ 
│                     │
└─────────────────────┘

첨부파일 [파일 선택]

□ 게시글 등록에 동의합니다.

            [등록] [취소]
